### 알게 된 것 / 삽질

- **`is_visiting_brewery` flag 가 DB 에 0건**: 더술닷컴 raw 데이터에 이 정보가 없거나,
  import 로직이 그 칼럼 안 채움. 더술닷컴의 visitingBrewery 페이지 (CLAUDE.md §13)
  별도 크롤링 필요 → 다음 세션 후보 (~30~40분).
- **`breweries.instagram` 칼럼도 0건**: 같은 import 한계. 양조장별 IG 핸들 별도
  수집 필요. 일단 cohort 스크립트는 `instagram` flag 점수만 무력화된 채 동작.
- **`as const` literal narrowing 우회 = `Widen<T>` trick**: ko 사전 `as const` 면
  string literal 까지 좁혀져서 en 의 다른 문자열이 type 안 맞음. recursive `Widen<T>`
  로 leaf string 을 `string` 으로 widen + nested object 는 그대로. DotPath 추출은
  여전히 정확.
- **Featured Brewery `.single()` 0건 silent fail**: `is_visiting_brewery=true`
  match 0건일 때 `.single()` 이 throw → 페이지 통째로 안 보이는 게 아니라, 코드
  destructure 가 silent 하게 null 받고 conditional render 가 영역 사라지게 함.
  진단하려면 직접 로그 또는 SSR debug 필요. 다음에 비슷한 패턴 = `.maybeSingle()`
  + fallback 명시.
- **Kakao DM 이 응답률 3~5배**: 한국 양조장 사장님 (5060대) 도 IG 매일, 이메일은
  주 1회. 화·수 오전 10~11시 발송이 베스트.
- **Next 16 `redirect()` 외부 URL 지원**: (5-01 발견 이후) 이번엔 dev server 도 turbopack
  fast-refresh 가 hero overhaul 같은 큰 변경도 5초 내 리프레시. 별도 서버 재시작 X.
